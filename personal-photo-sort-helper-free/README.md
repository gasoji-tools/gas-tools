# Personal Photo Sort Helper Free v1.0.0

指定したGoogle Driveフォルダ**直下**の写真（画像ファイル）を、Googleスプレッドシートに一覧化する無料GASツールです。

「この写真どこに入れたっけ」と毎回スクロールする前に、まず1フォルダの中身を見える化するためのツールです。

## できること

- フォルダ直下の画像ファイルを以下の5列で一覧化します
  - ファイル名 / 作成日 / 種類 / サイズ(MB) / URL
- 作成日の新しい順に並びます
- なお「作成日」はGoogle Driveのファイル作成（登録）日時（getDateCreated）、つまりそのファイルがDriveに追加された日時であり、写真の実際の撮影日（EXIF）とは限りません
- スプレッドシートのメニューに「Photo Sort Helper」が追加され、ワンクリックで再実行できます

対象とする画像形式: JPEG / PNG / GIF / HEIC / HEIF / WebP / TIFF / BMP

## しないこと（安心ポイント）

- ファイルの**削除・移動・リネーム・権限変更は一切行いません**（読み取りのみ）
- 外部サービスへの送信は一切ありません。データはあなたのスプレッドシートに書き出されるだけです
- サブフォルダの中までは見ません（直下のみ）

## セットアップ（所要3分）

1. 新しいGoogleスプレッドシートを開く（既存のものでも可）
2. メニューの「拡張機能」→「Apps Script」を開く
3. 最初から入っているコードを全部消して、`main.gs` の中身を貼り付ける
4. コード上部の `FOLDER_ID = 'YOUR_FOLDER_ID'` を自分のフォルダIDに書き換える
   - フォルダIDは、ブラウザでそのフォルダを開いたときのURL
     `https://drive.google.com/drive/folders/XXXXX` の `XXXXX` の部分です
5. 保存（Ctrl+S / Cmd+S）して、関数の選択で `listPhotos` を選び「実行」を押す
6. 初回は「承認が必要です」と出ます。自分のGoogleアカウントを選び、
   「詳細」→「（プロジェクト名）に移動」→「許可」の順に進めてください
   - この承認は「あなた自身のアカウントが、あなたのDriveとシートを読み書きする」ための承認です
7. スプレッドシートに「Photos」シートが作られ、一覧が出力されます

2回目以降は、スプレッドシート上部のメニュー「Photo Sort Helper」→「写真一覧を作成」で再実行できます。

## 出力イメージ

実際の出力も同じ列構成です。

| ファイル名 | 作成日 | 種類 | サイズ(MB) | URL |
|---|---|---|---|---|
| 沖縄_海.jpg | 2026/05/01 10:30 | JPEG | 3.42 | https://... |
| IMG_0123.heic | 2026/04/12 18:05 | HEIC | 2.10 | https://... |

同梱の `sample-output.csv`（出力例）で実際の出力イメージを確認できます。

## 制限事項（Free版）

- 対象は**1フォルダの直下のみ**です（サブフォルダの中は対象外）
- 複数フォルダ、タグ付け、重複候補の検出、月次レポート、CSV出力には対応していません
- 画像以外のファイル（動画・PDF・ドキュメント等）は対象外です（スキップされます）
- 写真が非常に多いフォルダ（数千枚規模）では実行に時間がかかることがあります

サブフォルダもまとめて整理したい場合は、Pro版（複数フォルダ・サブフォルダ再帰・タグ付け・重複候補の色分け・整理チェック・月次レポート・フォルダ別サマリ・CSV出力対応）をご検討ください。

## 注意

- 個人・家庭の写真整理向けツールです。法人の写真資産管理（DAM）や業務用の証憑管理システムではありません。
- 写真整理が必ず終わる・作業時間が減ることを保証するツールではありません。
- Google 公式 / 公認のツールではありません。

## よくあるエラー

- 「フォルダが開けません」: `FOLDER_ID` の値が間違っているか、そのフォルダへのアクセス権がありません
- 「設定が必要です」: `FOLDER_ID` が初期値 `YOUR_FOLDER_ID` のままです
