# Gasおじ GAS Tools

Gasおじラボが公開する、Google Apps Script向けの無料ツール集です。

## Included Tools

- `drive-file-list-free/`
  - Google Driveの指定フォルダ直下にあるファイルを、Googleスプレッドシートへ一覧化する無料GASです。
  - 詳しい使い方は `drive-file-list-free/README.md` を確認してください。
- `project-management-board-free/`
  - スプレッドシート上で使える、カンバン式のプロジェクト管理ボードです。
  - 公開note: <https://note.com/yuyukuma_gasoji/n/nd9df7532cfd0>
- `report-automation-free/`
  - スプレッドシートのデータから、簡易レポートを作成・送信する無料GASです。
  - 公開note: <https://note.com/yuyukuma_gasoji/n/nf18f5a6b4eaa>
- `helpdesk-management-free/`
  - 問い合わせチケットをスプレッドシートで管理する無料GASです。
  - 公開note: <https://note.com/yuyukuma_gasoji/n/ne2174cf20fe4>
- `attendance-management-free/`
  - 出退勤記録と月次集計をスプレッドシートで管理する無料GASです。
  - 公開note: <https://note.com/yuyukuma_gasoji/n/n74a4b8601591>
- `kanban-board-free/`
  - TODO / 進行中 / 完了の3レーンでタスクを管理する無料GASです。
  - 公開note: <https://note.com/yuyukuma_gasoji/n/n4886b0bbc637>
- `crm-management-free/`
  - 顧客一覧と商談履歴をスプレッドシートで管理する無料GASです。
  - 公開note: <https://note.com/yuyukuma_gasoji/n/n33ce00bffa76>
- `review-management-free/`
  - Googleマップ口コミなどの手動管理と返信テンプレート運用を補助する無料GASです。
  - 公開note: <https://note.com/yuyukuma_gasoji/n/n0be931ae9381>
- `mini-shop-order-tracker-free/`
  - 個人販売の注文をGoogleシート1枚で管理し、未発送・未入金を見える化する無料GASです。
  - 公開note: 準備中

## Held Out

- `reminder-automation-free` は、公開note候補と取得Gist実体が一致していないため、このrepo投稿準備から除外しています。
- 除外理由: Gistが非公開かつ「GASInvoice Pro - 請求書自動化ツール（購入者限定）」の実体だったため、無料ツールrepoへ混ぜません。

## Install / Download

npmから取得する場合:

```bash
npm install @gasoji/gas-tools
```

GitHubから確認する場合:

```bash
git clone https://github.com/gasoji-tools/gas-tools.git
```

## Notes

- このパッケージはGoogle公式・Google公認のツールではありません。
- Google Apps Scriptのコードは、利用者自身のGoogleアカウントとスプレッドシート上で実行してください。
- 共有してはいけないファイルや個人情報を含むフォルダでは、利用前に十分確認してください。
