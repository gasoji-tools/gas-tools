# Mini Shop Order Tracker Free（注文台帳 Free）

個人販売の注文を Google シート1枚で管理し、**未発送・未入金を見える化**するミニツールです。
BOOTH / BASE / メルカリ / 同人イベント / 手渡し / DM 注文など、バラバラになりがちな注文を1枚の台帳に戻します。

## できること

- `Orders` シート1枚で、注文日 / 購入者メモ / 商品名 / 金額 / 入金状態 / 発送状態 / 発送予定日 / メモ を一覧管理
- ボタン1つで「未発送一覧」「未入金一覧」をまとめた**自分宛てのリマインド下書き**を作成
- 自動送信はしません（下書きを作るだけ。見るだけでもOK）

## 使い方

公開note:
<https://note.com/yuyukuma_gasoji/n/n2f0f7ef81f79>

1. Google スプレッドシートを新規作成し、`拡張機能 > Apps Script` を開く
2. `main.gs` の内容を貼り付けて保存
3. シートを再読み込みすると、メニューに「注文台帳 Free」が表示されます
4. 「初期シートを作成」を実行 → `Orders` シートとサンプル注文ができます
5. サンプルを自分の注文に書き換える
   - 入金状態: `未入金` / `入金済み`
   - 発送状態: `未発送` / `梱包済み` / `発送済み`
   - 注文経路（BOOTH / DM など）は購入者メモ欄に書いておくと分かりやすいです
6. 「未発送・未入金リマインドを作成」を実行 → Gmail に自分宛ての下書きができます
7. 下書きを開いて、未発送・未入金の状況を確認

## 列の意味

| 列 | 内容 |
| --- | --- |
| 注文日 | 注文を受けた日（例 2026/06/17） |
| 購入者メモ | 相手の呼び名や経路など（例「BOOTH / にゃさん」）。個人情報は最小限に |
| 商品名 | 売った商品 |
| 金額 | その注文の合計金額（数値） |
| 入金状態 | `未入金` または `入金済み` |
| 発送状態 | `未発送` / `梱包済み` / `発送済み` |
| 発送予定日 | 発送予定の日付（空欄可）。超過すると下書きで強調されます |
| メモ | 配送方法など自由メモ |

## 注意

- 個人販売者向けの小さな注文台帳です。法人 EC の基幹受発注システムではありません。
- 氏名・住所・連絡先などの個人情報は、必要最小限を自分の責任で管理してください。
- 売上増加・発送ミスゼロ・法令遵守を保証するツールではありません。
- Google 公式 / 公認のツールではありません。
