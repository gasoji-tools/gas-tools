# DriveFileList Free

Google Driveの指定フォルダ直下にあるファイルを、Googleスプレッドシートへ一覧化する無料GASです。

「Driveであのファイルどこだっけ？」を減らすための小さな自動化です。月額ツールを入れるほどではないけれど、ファイル名・更新日・種類・URLを1枚で見たい人向けです。

## できること

- 1つのGoogle Driveフォルダを一覧化
- ファイル名、最終更新日、種類、URLをスプレッドシートへ出力
- 所有者メールを取得できる場合だけ表示
- スプレッドシートのメニューから手動更新

## 出力イメージ

sample-output.csv に、出力される列のサンプルを入れています。
公開時は実在しないサンプルURLとサンプルIDだけを使い、個人情報や内部ファイル名は入れません。

## Free版の制限

- サブフォルダの中は見ません
- 複数フォルダの同時管理はできません
- 条件フィルタ、CSV出力、ログはありません
- ファイルの削除、移動、権限変更、名前変更は行いません
- Google公式ツールではありません

この制限があるから、Free版は「まず1フォルダを見える化する」ためのものです。

## セットアップ

1. Googleスプレッドシートを新規作成します。
2. 拡張機能 → Apps Script を開きます。
3. main.gs の中身を貼り付けて保存します。
4. スプレッドシートに戻り、ページを再読み込みします。
5. 自動作成された 設定 シートの B2 にGoogle DriveのフォルダIDを入れます。
6. メニューの DriveFileList Free → ファイル一覧を更新 を押します。

初回実行時はGoogleの権限確認が出ます。表示内容を確認し、自分のDriveにアクセスする処理で問題ない場合だけ許可してください。

## フォルダIDの取り方

Driveで対象フォルダを開くと、URLは次の形になります。

https://drive.google.com/drive/folders/xxxxxxxxxxxxxxxxxxxx

folders/ の後ろの xxxxxxxxxxxxxxxxxxxx だけをコピーして、設定!B2 に貼り付けます。

## Pro版で外す予定の壁

Pro版は、Free版を使って「サブフォルダも見たい」「複数フォルダをまとめたい」「更新日や種類で絞りたい」と感じた人向けです。

- 複数フォルダ対応
- サブフォルダ再帰検索
- ファイル種別フィルタ
- 更新日フィルタ
- 重複ファイル候補
- ログシート
- 設定シート
- CSV出力
- 詳細セットアップガイド

## 注意

このGASはGoogle公式・Google公認のツールではありません。自分がアクセスできるフォルダだけを対象にし、共有してはいけないファイルや個人情報を含むフォルダでは利用前に十分確認してください。
